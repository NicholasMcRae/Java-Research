- What is JUnit5?
	- unlike previous versions, composed of several different modules
	- JUnit Platform + JUnit Jupiter + JUnit Vintage
		- Platform - foundation
		- Jupiter - programming and extension model
		- Vintage - old versions
- Supported Versions
	- Requires Java 8 (or higher) at runtime
